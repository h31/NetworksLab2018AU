# Описание протокола

Протокол поддерживает передачу двухбайтовых беззнаковых чисел и строк.
**Число** передается от младшего байта к старшему.
При передачи **строк** сначала пересылается её размер в виде двухбайтового числа, а затем сама строка побайтово.

Когда клиент подключается к серверу первое, что он должен прислать -- **строку**, которая будет интерпретирована как никнейм подключившегося пользователя. Далее любая присланная строка интерпретируется как сообщение от пользователя.
Сервер рассылает клиентам сообщения в следующем формате: часы [2 байта], минуты [2 байта], никнейм [строка], сообщение [строка].
